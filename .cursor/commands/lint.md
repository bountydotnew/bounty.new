Ultracite usage:
Usage: ultracite [options] [command]

Ship code faster and with more confidence.
Available subcommands: init, lint, format

Options:
  -V, --version                                   output the version number
  -h, --help                                      display help for command

Commands:
  init [options]                                  Initialize Ultracite in the current directory
  lint [specific files to lint...]                Run Biome linter without fixing files        
  format [options] [specific files to format...]  Run Biome linter and fixes files
  help [command]                                  display help for command
